# camera_recoder
My simple video recoder using OpenCV

## 주요 기능

1. **녹화 시작/종료**:  
   `Space` 키를 눌러 녹화 모드를 토글할 수 있습니다. 녹화가 시작되면 영상 화면에 빨간 원이 표시됩니다.

2. **흑백 모드 전환**:  
   `r` 키를 눌러 흑백 모드를 켜거나 끌 수 있습니다. 흑백 모드에서는 영상이 흑백으로 변환되어 녹화됩니다.

3. **녹화 시간 표시**:  
   녹화 중일 때, 화면에 녹화 시간이 초 단위로 표시됩니다.

4. **비디오 저장**:  
   녹화 중인 영상을 파일로 저장할 수 있습니다. 저장되는 파일 이름은 `recorded_video.avi`입니다.
   
6. **종료**:  
   `ESC` 키를 눌러 프로그램을 종료할 수 있습니다.
